WNXplorer

WNXplorer is a WordNet visualization application written in Java, and based on the Jung2 library.

Keyboard shortcuts

  • CTRL-c: clear the graph view.
  • ALT-s: display the search view.
  • ALT-i: display the synset properties view.
  • ALT-g: display the graph properties view.
  • p (when the graph view has the focus): entering/leaving the picking mode.
