WNXplorer

WNXplorer is a WordNet visualization application written in Java, and based on the Jung2 library.

http://njames.trevize.net/wiki/projects:wnxplorer


Usage

Make a search in the Search tab, and click using the left mouse button on a result (a synset) for adding it to the graph view.
You can develop a node (adding to the graph all his parent nodes) by using the right mouse button on a node and click the option "Develop".

The interaction with the graph is using a pan mode by default. Using the p key you enter/leave the picking mode that allow to pick a node and moving it.
When you click on a node using the left mouse button, you make appear all the information about the synset for this node in the Synset Info tab.

Keyboard shortcuts

  • CTRL-c: clear the graph view.
  • ALT-s: display the search view.
  • ALT-i: display the synset properties view.
  • ALT-g: display the graph properties view.
  • p (when the graph view has the focus): entering/leaving the picking mode.
  • F11: entering/leaving the fullscreen mode.
