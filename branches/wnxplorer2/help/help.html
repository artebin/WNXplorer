First start

The first time you launch WNXplorer, the application ask to you to indicate the location of the dict folder of a WordNet distribution. If you want to modify this location after the first launch of the application, you can directly modify the WNXplorer.properties file and update the value of the key PROPERTY_WORDNET_PATH.

Search

  • Make a search in the Search view. A left-click on a result (i.e. a SynSet) adds a node for this result on the graph view.

The graph view

  • There are two modes for interacting with the graph view. The modes are exclusive:
    • Pan mode: to translate de view
    • Picking mode: to pick and move nodes. Use the p key you enter/leave the picking mode
    You can always pan the view by using the middle mouse button and a click & drag.
  • Zoom operations: use the wheel mouse.
  • Nodes in the graph have 2 states:
    • selected: only one node at a time can be in the selected state. You can set the selected node by a left-click over a node when the graph view is in the pan mode. The selected node is described in the Description View. A selected node appears with a bold shape like in the following screenshot:
    • picked: a picked node can be move. To pick a node you can make a left-click over a node when the graph view is set in the Picking mode. You can move several nodes at the same time by setting the picked state on several nodes, to doing this: use the SHIFT+left click to pick each node. A picked node is filled in Yellow like in the following screenshot:
    Note that the two states are not exclusive, a node can be selected an picked.
  • each node has a contextual menu that appears with a right click over it.
  • to develop a node (i.e. add to the graph all its parent nodes directly accessible): (1) left double-click over a node, or (2) use the right mouse button on a node to make appear the contextual menu and click the option "Develop".
  • Clear the graph: CTRL-C

Keyboard shortcuts

  • CTRL-c: clear the graph view.
  • p (when the graph view has the focus): entering/leaving the picking mode.
  • F11: entering/leaving the fullscreen mode.
